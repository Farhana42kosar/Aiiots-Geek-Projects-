## Intelligent Document Summarization and Question Answering System

This project implements an intelligent NLP system capable of performing Document Summarization and Question Answering. The system processes large textual documents, generates concise summaries, and allows users to ask questions related to the document to retrieve relevant answers.

The goal of this project is to make large documents easier to understand by summarizing key information and enabling quick information retrieval through natural language questions.

## Features 
- Automatic document summarization
- Question answering based on document content
- Natural Language Processing (NLP) based text analysis
- Extractive summarization
- Context-aware question answering
- Efficient processing of long textual inputs

## How It Works

- The user provides a document as input.
- The system preprocesses the document by cleaning and tokenizing the text.
- Important sentences are identified to generate a summary of the document.
- The summarized content helps users quickly understand the document.
- Users can ask questions related to the document.
- The system analyzes the context and retrieves the most relevant answer.
